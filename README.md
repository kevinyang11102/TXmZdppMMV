# 个人博客系统简介 python194

## 项目概述

个人博客系统是基于 Django 框架开发，前端使用 Vue 技术进行构建，支持用户发表、管理和分享博文，同时集成了博文打赏、留言板等丰富的社交功能。

## 技术栈

- Python 3.7
- Django
- MySQL 5.7
- Vue

## 功能模块

### 系统角色

- 管理员
- 博主
- 游客

### 管理员功能

管理员可进行以下操作：

1. 查看系统首页
2. 管理个人中心
3. 博主管理
4. 博客文章管理
5. 标签类型管理
6. 文章类型管理
7. 博文排行管理
8. 博文打赏管理
9. 博文归档管理
10. 留言板管理
11. 博文分享
12. 系统管理

### 博主功能

博主可进行以下操作：

1. 系统首页管理
2. 个人中心管理
3. 博客文章管理
4. 博文打赏管理

### 游客功能

游客可以：

1. 查看系统首页
2. 阅读博客文章
3. 查看博文排行
4. 查看博文分享
5. 阅读公告信息
6. 浏览留言板

## 系统运行环境

- Python 3.7及以上版本
- MySQL 5.7及以上版本
